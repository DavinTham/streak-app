# 🔥 Streak

A gamified daily habit tracker for macOS. Keep **one** productive task alive with a streak, a satisfying complete button, a cozy reward sound, and a menu-bar item that shows your live streak count.

## Download

**[⬇️ Download the latest release](https://github.com/DavinTham/streak-app/releases/latest)**

- **Apple Silicon (M1/M2/M3/M4):** grab `Streak_x.y.z_aarch64.dmg` from the release.

## Install

1. Open the `.dmg` and drag **Streak** into your **Applications** folder.
2. The app isn't code-signed yet, so macOS Gatekeeper blocks the first launch. To open it:
   - **Right-click** `Streak.app` → **Open** → **Open**, or
   - Run: `xattr -dr com.apple.quarantine /Applications/Streak.app`

Requires macOS 10.15 or later.

## Features

- One daily task with a streak counter and best-streak tracking
- Big, tactile complete button with smooth, native-feeling animations
- A living, Duolingo-style flame that celebrates each completion
- macOS menu-bar item showing your current streak, with a quick-complete popover
- Evening reminder notification (6:30–7:30 PM) if you haven't completed yet
- Cozy reward sound on completion, with an on/off toggle
- Anti-cheat: one increment per day, and time-travel detection
